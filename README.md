# goodcoffee-dev
Dev environment for the GoodCoffee app

To start:

`git clone --recursive https://github.com/GoodNetwork/goodcoffee-dev.git`

`vagrant up`

`cd goodcoffee`

`git pull`

`npm install`

`php composer.phar install`

`webpack`

Important:
Add `192.168.56.101 goodcoffee.dev` to your /etc/hosts file
